JAC-RMIT-OSP
============

Building it is currently a 3 stage process

1. cd to "shared" and run "make"
2. cd to "client" and run "make"
3. cd to "server" and run "make"


as for running, the server is run with 
"server <portNo>"
and the client with
"client <serverAddress> <portNo> "
